## How to use

### Running Backend
- Change directory
    ```
    cd Backend/
    ```
- Create a env file in Backend folder like simple_env
- Install node module
    ```
    npm i
    ```
- Run server
    ```
    npm run dev
    ```
### Running Frontend
- Change directory
    ```
    cd Frontend/
    ```
- Create a env file in Backend folder like simple_env
- Install node module
    ```
    npm i
    ```
- Run server
    ```
    npm run dev
    ```
